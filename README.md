# c-browser
A modern, secure gateway for developers to access verified C, C++, and NASM compilers.

### Features
- **Architecture Detection:** Automatically sniffs for **Win64**, macOS, or Linux.
- **Security:** Integrated Google reCAPTCHA v2 to prevent bot scraping.
- **Minimalist UI:** Clean, dark-mode interface designed for developers.

### Tech Stack
- HTML5, CSS3 (Modern Dark Theme)
- JavaScript (OS Sniffing & API Integration)
- Google reCAPTCHA API
